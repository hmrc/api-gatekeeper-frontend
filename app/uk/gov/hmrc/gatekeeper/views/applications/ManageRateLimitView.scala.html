Change the rate limit tier for this application

! Warning: When you change the rate limit, you remove any deprecated API subscriptions.
Contact the application’s administrator. For further details, refer to the Lifecycle and Versioning page.

Select a new rate limit tier

Rate limits control how many requests your application can make in a given period. Choose the tier that matches the application’s expected traffic. If you need to increase the rate limit beyond the standard tiers, follow the guidance in the API Platform Rate Limit Increase Process page.

Note: Platinum and Rhodium rate limits are higher than the MDTP WAF (ISC) limit. To use them, the API Platform team must disable the MDTP WAF (ISC) limit for a known IP address or IP range.
Contact the API Platform team on #team-api-platform-sup or raise an APSR Jira ticket.
